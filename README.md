# strix

Strix Cunicularia (burrowing owl), an LLM powered RC car controller

## Hardware

- [Hardware wiring guide](docs/hardware-wiring.md) — wire the ESC and steering servo to the Arduino Nano (beginner friendly)
- [Controller takeover](docs/controller-takeover.md) — workaround for sealed receiver+ESC combos (e.g. Hosim X15W): automate a second controller's pots instead of wiring the ESC

## VLM setup guides

- [On-device (offline) VLM](docs/on-device-vlm.md) — run Gemma 3n on the phone itself
- [Self-hosted VLM server](docs/self-hosted-vlm-server.md) — serve a faster model from your own PC (Ollama + Caddy/Tailscale)
